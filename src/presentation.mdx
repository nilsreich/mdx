## Python Code

<CodeBlock language="python" title="fibonacci.py">
{`def fibonacci(n):
    """Generiere Fibonacci-Folge bis n Terme."""
    a, b = 0, 1
    result = []
    
    for _ in range(n):
        result.append(a)
        a, b = b, a + b
    
    return result

# Erste 10 Fibonacci-Zahlen
fib = fibonacci(10)
print(f"Fibonacci: {fib}")`}
</CodeBlock>

plt.grid(True)
